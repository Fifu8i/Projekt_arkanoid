# Projekt_arkanoid
Projekt na informatyka II
Gra arkanoid prosty działa (piłeczka odbija się od bloków oraz paletki. Bloki zmieniają kolory/życia w zależnosci od ilości trafień piłeczki). 
Gra posiada prosty mechanizm menu, w którym można sterowac strzałkami lub "w" oraz "s". Górny prostokąt oznacza wznowienie gry, a dolny zamkniecie gry (należy potwierdzić wybór opcji klikając "enter").
